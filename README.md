# Sathwin Reddy Julakanti - Portfolio

Personal portfolio website showcasing software engineering projects, technical skills, and professional experience.

## Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS v3
- **Animations:** Framer Motion
- **Fonts:** Geist Sans & Geist Mono

## Quick Start

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Deploy
npm start
```

Visit [http://localhost:3000](http://localhost:3000)

## Features

- Animated profile image with gradient effects
- Project showcase with preview images
- Technology stack visualization
- Professional experience timeline
- Responsive design optimized for all devices
